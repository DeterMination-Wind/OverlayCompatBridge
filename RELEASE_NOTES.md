# OverlayCompatBridge Release Notes

## v1.0.0

### 中文

- 首个公开版本。
- 为普通 Mindustry 客户端提供 OverlayUI 兼容层，让支持该接口的模组可以继续显示悬浮窗口。
- 提供统一的窗口管理器、拖动、缩放、固定、开关和位置记忆体验。
- 齿轮按钮固定在屏幕左侧中部，默认可通过按钮或 `Z` 快捷键打开。

### English

- First public release.
- Adds an OverlayUI compatibility layer for regular Mindustry clients so compatible mods can continue to show floating windows.
- Provides a shared window manager with drag, resize, pin, toggle, and saved layout behavior.
- The gear button is fixed at the left-middle side of the screen and can be opened by the button or the default `Z` hotkey.
